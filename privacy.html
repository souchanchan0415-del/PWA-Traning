プライバシーポリシー

最終更新日：2025-11-02 / 本ページはオフラインでも表示できるようプリキャッシュされます。

1. 基本方針

Train Punch（以下「本アプリ」）は、ユーザーのトレーニング記録を端末内（主に IndexedDB・LocalStorage・Cache Storage）に保存します。開発者のサーバーへ自動送信される個人データはありません。

本アプリはアクセス解析や行動トラッキングを行いません。

2. 収集・保存する情報（端末内）

  • トレーニング記録：種目・重量・回数・RPE・日付・メモ
  • アプリ設定：テーマ、タイマー、ウォッチ種目、通知可否 等
  • バックアップ/復元に使用するファイル（JSON/CSV）：ユーザー操作時のみ

保存期間はユーザーが削除するまでです（設定 → 全データ削除 やブラウザのサイトデータ削除で消去されます）。

3. 端末の権限・機能の利用

  • 通知：休憩タイマー等の通知に OS の許可が必要です。通知内容は端末内で生成・表示され、サーバー送信は行いません。
  • 音声再生：beep.wav を再生します。録音・マイクアクセスは行いません。
  • クリップボード：メールアドレスやテンプレ文面のコピー時に使用します（ユーザー操作時のみ）。
  • ファイルアクセス：JSON/CSV のインポート時にローカルファイルを読み込みます（ブラウザ内で完結）。
  • PWA/オフライン：Service Worker が静的アセットをキャッシュし、オフライン動作と高速化を行います。

4. 外部サービスと広告

本アプリ本体は広告を表示しませんが、支援ページ（support.html）では運営費捻出のため広告（例：Google AdSense）やアフィリエイトリンクを掲載する場合があります。

  • 広告配信にはクッキー等が利用されることがあります。詳細は各事業者のポリシーをご確認ください。
  • EEA/英国/スイス等からのアクセスでは、Google の同意管理によりクッキー等の利用可否を確認するメッセージが表示される場合があります。
  • 投げ銭や月額支援リンク（例：Ko-fi / GitHub Sponsors）を設置することがあります。決済は各サービス上で行われ、本アプリはクレジットカード番号等の決済情報を保持しません。

※外部サイトのご利用は各サイトのポリシーが適用されます。

5. データのエクスポート/削除

バックアップ
設定 → JSONエクスポート / CSVエクスポート で出力できます。
復元
JSONインポート / CSVインポート から端末内に読み込みます。
削除
設定 → 全データ削除 で記録を消去できます。ブラウザ側のサイトデータ削除でも消去されます。

6. セキュリティ

端末内で完結する設計のため、サーバー漏えいリスクはありません。端末の画面ロック・OS/ブラウザの最新化・共有端末での利用回避など、基本的な対策を推奨します。

7. 児童の利用について

本アプリは一般ユーザー向けです。13歳未満の方は、保護者の同意なく個人情報を送信しないでください（本アプリはサーバー送信を行いませんが、連絡等を行う場合の一般的な注意喚起として記載しています）。
